HTML Tutorial

HTML (HyperText Markup Language) is the foundation of every web page. It provides the structure and defines the content through a series of elements.

Basic Document Structure:
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First HTML Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
Common HTML Elements:
  • <h1> - <h6>: Headings
  • <p>: Paragraphs
  • <a>: Hyperlinks
  • <img>: Images
  • <ul> & <ol>: Lists
  • <div>: Generic container

Experiment with these elements to build your own web pages. HTML is all about nesting tags and adding attributes to enhance functionality.

CSS Tutorial

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the visual appearance of HTML elements, including colors, fonts, and spacing.

Basic CSS Syntax:
/* CSS syntax example */
selector {
  property: value;
}
Example - Styling a Page:
body {
  background-color: #f8f9fa;
  font-family: Arial, sans-serif;
}

h1 {
  color: #007bff;
}

p {
  line-height: 1.6;
}

CSS can be applied inline, embedded in the head section using a <style> tag, or through an external stylesheet. Experiment with selectors, properties, and values to see immediate changes in your page layout.

Advanced Topics:
  • Responsive Design with Media Queries
  • Flexbox and Grid for layouts
  • Animations and Transitions

JavaScript Tutorial

JavaScript is a powerful programming language that enables dynamic behavior on websites. It allows you to manipulate the DOM, handle events, and create interactive user experiences.

Basic JavaScript Example:
// JavaScript example: Display an alert when the document loads
document.addEventListener('DOMContentLoaded', function() {
  alert('Welcome to JavaScript!');
});
Working with Variables and Functions:
// Variables and functions in JavaScript
let greeting = 'Hello, World!';
function sayHello(name) {
  return greeting + ' ' + name;
}
console.log(sayHello('Student'));

JavaScript can be included directly in your HTML using the <script> tag, or through an external file. It is essential for adding interactivity to your website.

Advanced Topics:
  • DOM Manipulation
  • Event Handling
  • Asynchronous Programming (Promises, Async/Await)
  • APIs and AJAX
